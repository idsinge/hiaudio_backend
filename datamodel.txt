* index page: list of compositions
* composition page: list tracks

* file structure:
- users/
  - <userid>
    - userinfo/
    - collections/
      - <collectionid>/
    - compositions/
      - <compositionid>/
        - tracks/
          - <track filename>
        - contributors
          - <contributor userid>


User:
 - id  
 - uid

UserInfo:
 - id
 - user_id (duplicated) 
 - name
 - profile_pic
 - google_uid
 - google_name
 - google_email
 - google_profile_pic

Collection:
- id
- uuid
- privacy
- title
- user_id
- parent_id
- compositions

Composition:
 - id
 - uuid
 - user_id
 - title
 - collection_id
 - privacy (1=Public, 2=Only Reg, 3=Private)
 - opentocontrib
 - contributors
 - tracks

Track:
 - id
 - uuid
 - composition_id
 - title
 - user_id
 - path

Contributor:
 - id
 - user_id
 - user_uid
 - composition_id
 - role (0=None, 1=Owner, 2=Admin, 3=Member, 4=Guest)
