

* index page: list of compositions
* composition page: list recordings

* file structure:
- users/
  - <userid>
    - compositions/
      - <compositionid>/
        - recordings/
          - <recording filename>


User:
 - id
 - username 
 - groups
 - name
 - email
 - profile_pic
 - compositions

Composition:
 - id
 - userid
 - title
 - perms
 - recordings

Recording:
 - id
 - compositionid
 - title
 - author? userid?
 - filepath


---

Perm:
 - id
 - userid   [null=wildcard]
 - groupid  [null=wildcard]
 - compositionid
 - read?
 - write?

Group:
 - id
 - name
