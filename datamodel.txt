* index page: list of compositions
* composition page: list tracks

* file structure:
- users/
  - <userid>
    - userinfo/
    - compositions/
      - <compositionid>/
        - tracks/
          - <track filename>
        - contributors
          - <contributor userid>


User:
 - id 
 - name
 - uid
 - profile_pic
 - compositions
 - userinfo

UserInfo:
 - id 
 - google_uid
 - google_name
 - google_email
 - google_profile_pic

Composition:
 - id
 - userid
 - title
 - privacy (1=Public, 2=Only Reg, 3=Private)
 - contributors
 - tracks

Track:
 - id
 - compositionid
 - title
 - path

Contributor:
 - id
 - userid
 - role (1=Owner, 2=Admin, 3=Member, 4=Guest)

---

Perm:
 - id
 - userid   [null=wildcard]
 - groupid  [null=wildcard]
 - compositionid
 - read?
 - write?

Group:
 - id
 - name
