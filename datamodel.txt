

* index page: list of compositions
* composition page: list tracks

* file structure:
- users/
  - <userid>
    - compositions/
      - <compositionid>/
        - tracks/
          - <track filename>


User:
 - id
 - username 
 - groups
 - name
 - email
 - profile_pic
 - compositions

Composition:
 - id
 - userid
 - title
 - privacy
 - perms
 - tracks

Track:
 - id
 - compositionid
 - title
 - author? userid?
 - filepath


---

Perm:
 - id
 - userid   [null=wildcard]
 - groupid  [null=wildcard]
 - compositionid
 - read?
 - write?

Group:
 - id
 - name
