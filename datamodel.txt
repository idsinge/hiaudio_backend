* index page: list of compositions
* composition page: list tracks

* file structure:
- users/
  - <userid>
    - userinfo/
    - collections/
      - <collectionid>/
    - compositions/
      - <compositionid>/
        - tracks/
          - <track filename>
        - contributors
          - <contributor userid>

- verificationcode

User:
 - id  
 - uid
 - is_admin
 - terms_accepted

UserInfo:
 - id
 - user_id (duplicated) 
 - name
 - profile_pic
 - user_uid 
 - user_email

Collection:
- id
- uuid
- privacy
- title
- description
- user_id
- parent_id
- compositions

Composition:
 - id
 - uuid
 - user_id
 - title
 - description
 - collection_id
 - privacy (1=Public, 2=Only Reg, 3=Private)
 - opentocontrib
 - contributors
 - tracks

Track:
 - id
 - uuid
 - composition_id
 - title
 - user_id
 - user_uid
 - path

Contributor:
 - id
 - user_id
 - user_uid
 - composition_id
 - role (0=None, 1=Owner, 2=Admin, 3=Member, 4=Guest)

VerificationCode:
 - email
 - code 
 - date 
 - attempts
